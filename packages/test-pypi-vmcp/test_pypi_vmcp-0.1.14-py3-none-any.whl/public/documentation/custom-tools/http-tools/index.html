HTTP Tools

HTTP tools allow you to make API calls to external services with support for various authentication methods.

Basic Structure​

{
  "name": "get_health",
  "description": "Get server health",
  "tool_type": "http",
  "api_config": {
    "method": "GET",
    "url": "https://api.example.com/health",
    "headers": {},
    "query_params": {},
    "auth": {
      "type": "none"
    }
  },
  "variables": []
}

HTTP Methods​

  • GET: Retrieve data
  • POST: Create resources
  • PUT: Update resources
  • DELETE: Remove resources
  • PATCH: Partial updates

Authentication Types​

None​

{
  "auth": {
    "type": "none"
  }
}

API Key (Header)​

{
  "auth": {
    "type": "apikey",
    "apiKey": "your-api-key",
    "keyName": "X-API-Key"
  }
}

Bearer Token​

{
  "auth": {
    "type": "bearer",
    "token": "your-bearer-token"
  }
}

Basic Auth​

{
  "auth": {
    "type": "basic",
    "username": "user",
    "password": "pass"
  }
}

Dynamic Parameters​

URL Parameters​

{
  "name": "get_user",
  "api_config": {
    "method": "GET",
    "url": "https://api.example.com/users/@param.user_id"
  },
  "variables": [
    {
      "name": "user_id",
      "type": "str",
      "required": true
    }
  ]
}

Query Parameters​

{
  "api_config": {
    "method": "GET",
    "url": "https://api.example.com/search",
    "query_params": {
      "q": "@param.query",
      "limit": "@param.limit"
    }
  }
}

Headers​

{
  "api_config": {
    "headers": {
      "Content-Type": "application/json",
      "X-Custom-Header": "@param.value"
    }
  }
}

Request Body (POST/PUT)​

{
  "api_config": {
    "method": "POST",
    "url": "https://api.example.com/users",
    "body": {
      "name": "@param.name",
      "email": "@param.email"
    }
  }
}

Using Environment Variables​

Store sensitive data in environment variables:

{
  "api_config": {
    "url": "@config.api_url",
    "auth": {
      "type": "apikey",
      "apiKey": "@config.api_key",
      "keyName": "X-API-Key"
    }
  },
  "environment_variables": ["api_url", "api_key"]
}

Complete Examples​

REST API Call​

{
  "name": "create_task",
  "description": "Create a new task",
  "tool_type": "http",
  "api_config": {
    "method": "POST",
    "url": "https://api.taskmanager.com/tasks",
    "headers": {
      "Content-Type": "application/json"
    },
    "auth": {
      "type": "bearer",
      "token": "@config.api_token"
    },
    "body": {
      "title": "@param.title",
      "description": "@param.description",
      "priority": "@param.priority"
    }
  },
  "variables": [
    {
      "name": "title",
      "type": "str",
      "required": true
    },
    {
      "name": "description",
      "type": "str",
      "required": false
    },
    {
      "name": "priority",
      "type": "int",
      "required": false,
      "default_value": 1
    }
  ],
  "environment_variables": ["api_token"]
}

Weather API​

{
  "name": "get_weather",
  "description": "Get weather for a city",
  "tool_type": "http",
  "api_config": {
    "method": "GET",
    "url": "https://api.weather.com/v1/weather",
    "query_params": {
      "city": "@param.city",
      "units": "metric"
    },
    "auth": {
      "type": "apikey",
      "apiKey": "@config.weather_api_key",
      "keyName": "X-API-Key"
    }
  },
  "variables": [
    {
      "name": "city",
      "type": "str",
      "required": true
    }
  ],
  "environment_variables": ["weather_api_key"]
}

Response Handling​

The tool returns the HTTP response including:

  • Status code
  • Headers
  • Body (JSON parsed if applicable)

Best Practices​

  1. Use Environment Variables: Never hardcode API keys
  2. Error Handling: Check status codes
  3. Timeouts: APIs have default 30s timeout
  4. Rate Limiting: Be mindful of API limits
  5. HTTPS Only: Always use secure connections

Troubleshooting​

401 Unauthorized​

  • Check authentication credentials
  • Verify API key is correct
  • Ensure token hasn't expired

404 Not Found​

  • Verify URL is correct
  • Check parameter substitution

429 Too Many Requests​

  • Implement rate limiting
  • Add delays between calls
