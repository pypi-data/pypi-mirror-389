Python Tools

Python tools allow you to execute Python code with full access to Python's standard library. Perfect for data processing, calculations, and complex logic.

Basic Structure​

{
  "name": "add_numbers",
  "description": "Add two numbers",
  "tool_type": "python",
  "code": "def main(a: int, b: int):\n    return a + b",
  "variables": [
    {
      "name": "a",
      "description": "First number",
      "type": "int",
      "required": true
    },
    {
      "name": "b",
      "description": "Second number",
      "type": "int",
      "required": true
    }
  ]
}

Supported Types​

String (str)​

def main(name: str, greeting: str):
    return f"{greeting}, {name}!"

Integer (int)​

def main(x: int, y: int):
    return x * y

Float​

def main(radius: float):
    import math
    return math.pi * radius * radius

List​

def main(numbers: list[int]):
    return sum(numbers) / len(numbers)

Dictionary​

def main(data: dict):
    keys = list(data.keys())
    return f"Keys: {', '.join(keys)}"

Boolean​

def main(is_active: bool):
    return "Active" if is_active else "Inactive"

Complete Examples​

Data Processing​

{
  "name": "process_sales_data",
  "description": "Calculate total sales and average",
  "tool_type": "python",
  "code": "def main(sales: list[float]):\n    total = sum(sales)\n    average = total / len(sales) if sales else 0\n    return {\n        'total': round(total, 2),\n        'average': round(average, 2),\n        'count': len(sales)\n    }",
  "variables": [
    {
      "name": "sales",
      "description": "List of sale amounts",
      "type": "list",
      "required": true
    }
  ]
}

Text Processing​

def main(text: str, operation: str):
    if operation == "upper":
        return text.upper()
    elif operation == "lower":
        return text.lower()
    elif operation == "title":
        return text.title()
    elif operation == "reverse":
        return text[::-1]
    return text

Complex Logic​

def main(n: int):
    """Calculate Fibonacci number"""
    if n <= 0:
        return "Invalid input"
    elif n == 1:
        return 0
    elif n == 2:
        return 1
    else:
        a, b = 0, 1
        for _ in range(n - 2):
            a, b = b, a + b
        return b

Error Handling​

def main(numerator: float, denominator: float):
    if denominator == 0:
        raise ValueError("Cannot divide by zero")
    return numerator / denominator

Default Values​

{
  "variables": [
    {
      "name": "title",
      "type": "str",
      "required": false,
      "default_value": "Mr/Ms"
    }
  ]
}
def main(name: str, title="Mr/Ms"):
    return f"Hello {title} {name}!"

Best Practices​

  1. Type Hints: Always use type hints
  2. Error Handling: Handle edge cases
  3. Return Values: Return JSON-serializable data
  4. Keep It Simple: One responsibility per tool
  5. Document: Clear descriptions

Limitations​

  • No external packages (stdlib only)
  • Execution timeout: 30 seconds
  • Memory limit: 256MB
  • No file system access
  • No network access
