Configuration
=============

**Configure:**

#. Go to *Timesheets* > *Configuration* > *Predefined Descriptions*.
#. Click on *Create*.
#. Enable *Allow Free Text* to see all additional options, *Append Free Text* and *Require Free Text*.
#. Save the description.
#. When creating a timesheet, select the predefined description to automatically fill or append text to the entry.

Usage
=====

To use this module, go to Timesheets > Configuration > Predefined Descriptions.

Create or edit a predefined description and configure:

#. Allow Free Text: lets users add additional text.
#. Free Text Required: forces users to fill in additional text.
#. Append Free Text to Name: appends the additional text to the description name.

These features only work if the analytic line is linked to a task or ticket.
