# Pyramid Welcome Template

A bold, responsive landing page for Pyramid projects generated by Projex CLI.

## Colors
- Primary: `#CD2027`
- Secondary: `#A91C22`
- Accent: `#FF4757`
- Background: Light `#F5F5F5` / Dark `#1F1F1F`

## Integration
See `integration.py` for a minimal `main` that mounts the root route.

## Customize
- Replace icons and text in feature cards
- Edit CSS variables for brand tweaks

## Assets
- Inter (Google Fonts), Font Awesome (CDN)
