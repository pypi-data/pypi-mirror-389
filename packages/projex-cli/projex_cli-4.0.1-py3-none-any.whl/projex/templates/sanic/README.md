# Sanic Welcome Template

A vibrant, responsive landing page for Sanic API projects generated by Projex CLI.

## Colors
- Primary: `#FF0080`
- Secondary: `#CC0066`
- Accent: `#FF4DA6`
- Background: Light `#FFFFFF` / Dark `#1C1C1C`

## Integration
Return the `welcome.html` at `/` and a JSON payload at `/api` (see `integration.py`).

## API Explorer
Sanic does not ship with Swagger by default; use `sanic-ext` to enable OpenAPI docs if desired.

## Customize
- Update feature cards and links
- Tweak `:root` variables for branding
